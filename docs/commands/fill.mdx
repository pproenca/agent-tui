title: fill
description: 'Fill an input field with text'
---

## Usage

```bash
agent-tui fill <ref> <value>
```

## Description

Fills an input field identified by its element ref with the specified text value.

## Arguments

| Argument | Description |
|----------|-------------|
| `<ref>` | Element ref from snapshot (e.g., `@e1`) |
| `<value>` | Text value to fill |

## Examples

```bash
# Fill a project name input
agent-tui fill @e1 "my-app"

# Fill with spaces
agent-tui fill @e1 "My Awesome Project"

# Fill email field
agent-tui fill @e2 "user@example.com"
```

## Workflow

```bash
# 1. Take snapshot to see elements
agent-tui snapshot -i
# Output: @e1 [input:name] "" (3,25) *focused*

# 2. Fill the input
agent-tui fill @e1 "my-value"

# 3. Verify the change
agent-tui snapshot -i
# Output: @e1 [input:name] "my-value" (3,25) *focused*
```

## Notes

<Warning>
  Always take a fresh snapshot before using `fill`. Element refs change when the UI updates.
</Warning>

<Info>
  `fill` replaces the entire field content. To append text, use `type` instead.
</Info>
