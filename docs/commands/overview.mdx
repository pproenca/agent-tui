## Quick Reference

| Command | Description | Example |
|---------|-------------|---------|
| [`spawn`](/commands/spawn) | Start TUI app | `agent-tui spawn htop` |
| [`snapshot`](/commands/snapshot) | Get screen state | `agent-tui snapshot -i` |
| [`fill`](/commands/fill) | Fill input field | `agent-tui fill @e1 "text"` |
| [`click`](/commands/click) | Click element | `agent-tui click @e1` |
| [`keystroke`](/commands/keystroke) | Send key | `agent-tui keystroke Enter` |
| [`type`](/commands/type) | Type text | `agent-tui type "hello"` |
| [`wait`](/commands/wait) | Wait for condition | `agent-tui wait "Ready"` |
| [`kill`](/commands/kill) | Terminate session | `agent-tui kill` |
| [`sessions`](/commands/sessions) | List sessions | `agent-tui sessions` |

## Global Options

All commands support these global options:

| Option | Description |
|--------|-------------|
| `--session <id>` | Target a specific session |
| `-f, --format <fmt>` | Output format: `text` or `json` |
| `-v, --verbose` | Enable verbose output |
| `--help` | Show help for command |

## JSON Output

Add `-f json` to any command for machine-readable output:

```bash
agent-tui snapshot -i -f json | jq '.elements'
agent-tui sessions -f json | jq '.sessions[0].id'
```
